JSONObject eu.h2020.symbiote.messaging.RpcServer.getResourcesUrls ( JSONObject  resourceIdList)

Spring AMQP Listener for providing resource urls. This method is invoked when a request for access to resources is made by an aplication/enabler. CoreInterface forwards a list of resource ids to CoreResourceAccessMonitor coming from the application/enabler and CoreResourceAccessMonitor responds with the urls of the specified resources.

Parameters
resourceIdListThe list of resource ids
Returns
The urls of the resources specified in the resourceIdList
